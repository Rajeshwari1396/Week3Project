# 🌸 Iris Flower Clustering Project

## 📌 Overview

This project demonstrates **Unsupervised Machine Learning** using the Iris Flower Dataset. The goal is to cluster flowers into groups using the K-Means algorithm without using their labels.

Principal Component Analysis (PCA) is used to reduce the dataset to two dimensions for visualization.

---

## 📂 Dataset

- Iris Dataset (UCI Machine Learning Repository / Kaggle)
- 150 Samples
- 4 Numerical Features
- 3 Flower Species

Features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Species:

- Setosa
- Versicolor
- Virginica

## 📊 Results

- Optimal K = 3
- PCA reduced 4 features to 2 principal components.
- K-Means successfully identified the three natural clusters.
